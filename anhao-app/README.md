
# 安好一日 (Anhao Yiri) 🌿

**“愿你今日安好。”**

这是一个基于 Google Gemini API 构建的温润生活助理应用。它不仅仅是一个 AI 工具，更是一个提供情绪陪伴、创意绘图和实时语音交流的私人港湾。

## ✨ 功能特性

- **今日安好 (Dashboard)**：治愈系的首页，提供每日箴言和功能入口。
- **心语空间 (Chat)**：温柔、富有同理心的 AI 对话伙伴，倾听你的每一个思绪。
- **绘梦板 (Image)**：将你的梦境或灵感通过 Gemini 2.5 Flash Image 转化为视觉艺术。
- **愈见声音 (Live)**：低延迟的实时语音交互，感受如老友般的耳边陪伴。

## 🛠️ 技术栈

- **前端**: React 19, Tailwind CSS
- **AI 模型**: Google Gemini API (Gemini 3 Flash, Gemini 2.5 Flash Image, Gemini 2.5 Flash Native Audio)
- **字体**: Noto Serif SC (思源宋体), Inter

## 🚀 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/你的用户名/anhao-yiri.git
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置 API Key**
   在你的环境变量中设置 `API_KEY`。

4. **启动项目**
   ```bash
   npm run dev
   ```
